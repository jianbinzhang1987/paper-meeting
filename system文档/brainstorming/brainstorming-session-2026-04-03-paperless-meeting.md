## SCAMPER 维度扫描

### S — 替代
- **[e-signin]** 电子签到系统 — 扫码/NFC/人脸三种签到，管理端实时统计
- **[digital-raise-hand-queue]** 数字举手/发言排队 — 悬浮按钮+队列管理+主持人优先级控制
- **[seamless-checkin]** 无感签到升级版 — WiFi/蓝牙信标自动识别

### C — 组合
- **[co-read-co-annotate]** ⭐ 共阅协注模式 — 主持人同步翻页 + 全员独立标注 + 一键共享
- **[annotation-classify]** 标注分类与定向分享 — 分类标签 + 定向分享范围
- **[anchor-discussion]** 锚点讨论 — 在材料某处发起讨论，内容与材料绑定

### A — 适配
- **[multi-role-UI]** 多角色智能界面 — 同一平板按角色切换（主持人/参会者/记录员）
- **[three-state-UI]** 会前/会中/会后三态切换 — 跟随会议生命周期变化界面

### M — 修改/放大
- **[material-deep-reading]** 材料深度阅读引擎 — 关键词高亮、摘要侧边栏、精读模式
- **[speech-heat-ranking]** 发言竞争热度排名 — 点赞排序代替简单排队
- **[large-conference-multiroom]** 万人大会/多会议室 — 主会场+分会场联动

### P — 其他用途
- **[opinion-survey-election]** 民意调研/选举引擎 — 独立于会议的投票选举平台
- **[knowledge-warehouse]** 企业知识沉淀仓 — 会议成果自动归档，新员工可回溯
- **[remote-interview]** 远程面试/答辩 — 复用会议核心交互

### E — 消除
- **[AI-meeting-minutes]** AI 自动生成会议纪要
- **[electronic-desk-sign]** 电子桌签 — 平板倒扣显示姓名+职务
- **[task-closed-loop]** 任务闭环追踪 — 会上标注责任人+截止日+到期提醒
- **[self-service-material]** 自助材料库 — 参会者自主浏览回看
- **[realtime-minutes]** 实时纪要伴随生成 — AI 实时起草，人审核确认

### R — 反转
- **[secure-material-sandbox]** 保密材料沙箱 — 屏幕水印+拍照溯源+会后自毁

### What If — 激进创新

- *What If 会议室是智能体？* — 不实用
- *What If 异步会议？* — 不符合场景
- *What If 保密？* — → [secure-material-sandbox]

### 六顶思考帽

- **白帽（事实）**：[smart-room-booking] 智能会议室预约、[meeting-templates] 会议模板引擎
- **红帽（情感）**：[welcome-homepage] 参会者欢迎首页、[reading-decompress] 材料阅读减负
- **黄帽（收益）**：共阅协注为核心卖点
- **黑帽（风险）**：[ai-minutes-human-review] AI纪要人工校对、[elder-mode] 长辈模式、[offline-resilience] 弱网/断网容灾
- **蓝帽（全局）**：功能模块地图梳理完成
