# 客户端联调与发布清单

## 自动化测试基线

- Flutter：执行 `flutter test`
- 后端模块编译：执行 `mvn -pl yudao-module-meeting -am compile "-Dmaven.test.skip=true"`
- 关键覆盖点：
  - 通知已读与置顶
  - 同屏角色切换
  - 无进行中会议
  - 文稿批注与书签增删
  - 密码必填校验
  - 离线浏览闸门

## 真机联调清单

### 1. 启动与签到

- 配置服务器、会议室、座位、设备名后重启应用，确认配置仍可恢复。
- 含会议密码的会议验证“会议密码必填”。
- 含个人密码的账号验证“个人密码必填”和错误密码拦截。
- 强制注销后确认回到签到页，并显示明确提示。

### 2. 文稿与资料

- 打开 PDF，验证翻页、跳页、缩放和最近访问。
- 新增/删除批注、书签后重启应用，确认本地仍可恢复。
- 同账号换设备登录，确认文稿标记可同步。
- 执行资料预加载，断网后确认缓存资源仍可打开。

### 3. 通知、服务与实时控制

- 通知分页加载、已读回写、未读数统计、置顶与自动轮播。
- 呼叫服务后检查状态流转：`pending -> accepted -> processing -> completed/canceled`。
- 同屏申请、审批、拒绝、强制返回、强制停止全链路验证。
- 视频同步验证打开、播放、暂停、拖动进度。
- 统一计时验证开始、停止、断线重连恢复。

### 4. 表决与签名

- 实名表决验证开始、结束、发布和统计刷新。
- 匿名表决验证：
  - 普通参会端在发布前不可见统计结果。
  - 主持人/秘书在发布前可见统计结果。
  - 发布后所有终端显示一致的汇总结果。
- 签名提交后验证服务端记录、客户端重复提交提示和最近一次签名回显。

### 5. 异常与恢复

- 启动时无会议页面。
- 接口失败页面重试。
- 实时断连页面切换“继续离线浏览”。
- 离线浏览后恢复联网，确认能重新进入在线状态并拉取实时快照。

## 发布流程

### 1. 发布前检查

- 确认数据库已执行 `paper-meeting-server/sql/mysql/meeting_schema_patch.sql`。
- 确认服务端静态文件、签名上传目录、WebSocket 路径配置可用。
- 确认 Android 安装包版本号、更新说明和下载地址。

### 2. 构建与冒烟

- 客户端执行一次 `flutter test`。
- 服务端执行一次 `mvn -pl yudao-module-meeting -am compile "-Dmaven.test.skip=true"`。
- 安装到至少 1 台平板执行启动、签到、文稿、通知、表决、签名、断网恢复冒烟。

### 3. 发布后回归

- 检查启动页配置恢复。
- 检查签到、通知分页、文稿标记同步。
- 检查视频同步、统一计时、服务呼叫状态更新。
- 检查匿名表决发布前后展示规则。
