# 无纸化会议系统管理端验收清单

## 1. 会前准备

- [ ] 新建会议：可填写会议名称、时间、会议室、审批方式、控制方式、保密级别。
- [ ] 预约审批：预约会议可进入审批流，审批后状态从“待审批”变为“已预约”。
- [ ] 会议模板：可通过模板创建会议，模板会议不会直接进入会中状态。
- [ ] 会议资料：可在会议详情中上传资料、删除资料、查看资料列表。
- [ ] 参会人：可单个添加参会人、删除参会人、导出签到表。
- [ ] 会议消息：可创建草稿消息、编辑消息、发布消息。
- [ ] 用户组邀请：会议管理流程支持按会议用户组批量导入参会人。

## 2. 会中控制

- [ ] 会议首页：可看到待审批、即将开始、会议室冲突等快捷告警。
- [ ] 表决控制中心：可切换进行中的会议，执行开始、结束、发布、导出和强制返回同屏。
- [ ] 消息阅读进度：通知列表能看到已读/未读人数，且可打开阅读明细。
- [ ] 会中服务：实时状态页可看到同屏申请、服务请求等会中事件。

## 3. 会后归档

- [ ] 已归档会议可导出资料。
- [ ] 已归档会议可导出表决结果。
- [ ] 已归档会议可导出签到表。
- [ ] 已归档会议可导出批注。

## 4. 资源与终端配置

- [ ] 公共资料库支持目录树浏览、上传录入、基础预览。
- [ ] 客户端样式支持新增、编辑、启用和实时预览。
- [ ] 安装包管理支持上传地址回填、版本概览、启用切换。
- [ ] 会议贴牌支持网站名称、Logo、侧边栏文案配置。

## 5. 权限与菜单

- [ ] 会议系统菜单可访问首页、我的会议、审批、会议列表、归档、模板、会议室、表决控制、会中消息、公共资料库、样式、安装包等页面。
- [ ] 会议详情相关接口权限包含 agenda、attendee、file、vote。
- [ ] 高风险操作需重点验证：发布消息、导出资料、导出表决、强制返回同屏、启用安装包、启用样式。

## 6. 建议验收顺序

1. 预约一场新会议并完成审批。
2. 配置参会人、资料、议题和消息。
3. 启动会议并在表决控制中心执行一次完整表决。
4. 使用客户端进入会议并验证消息已读、资料访问、表决提交。
5. 结束并归档会议，验证四类导出能力。
